# 交战与地块归属规则（最终版）

> **创建**: 2026-06-21
> **状态**: 经用户多轮纠正确认

---

## 规则0：出发归属地

每个师下令移动时记录 `origin_province`（出发地），移动全程不变。

用途：同出发地的后续进攻师 → 进预备队（时间线落后）。

---

## 规则1：交战判定（最高优先级）

**核心：只要两个不同国家的师在同一个省份，立刻开战。**
不管：谁的地盘、谁先到、移动中还是静止。

### 触发时机
1. **下令瞬间**：move_division 下令时查目标地块有无敌军师 → 有则立刻开战
   - 例：B 往省X进军，A 的 location 在省X（移动中）→ 下令瞬间就开战（X 上有A）
2. **每小时持续检查**：check_engagements 扫描所有移动中的师
   - 移动过程中敌军进入目标地块 → 立刻开战
3. **到达瞬间**：师到达某省，该省有敌军 → 开战（由 check_engagements 覆盖）

### 关键原则
- **进军即判定，不是到达才判定**
- 师的 location 在移动中是出发地（没变），所以"进军往省X"时，查省X的 location 上的师

---

## 规则2：地块归属

### 战斗期间
**战斗进行中，绝不改变任何地块归属。**
不管有没有到达目标地、地方有没有到达出发地。

### 战斗结束后
检查地块归属（两个途径）：

**途径A：战斗胜利占领（resolve 里处理）**
- 守方全退/歼灭（带溃）→ 攻方占领
- 攻方全退/歼灭 → 守方保住

**途径B：到达占领（advance_movement 里处理）**
到达某省时：
- 该省**有战斗进行中** → 不改变归属
- 该省**无战斗** + **有敌军师** → 不改变归属（规则1触发战斗）
- 该省**无战斗** + **无敌军** + **非己方** → 占领
- 该省**无战斗** + **无敌军** + **己方** → 无变化

## 规则3：状态机

每个师的状态：
```
静止 → 下令移动 → 移动中(记录origin)
  ├─ 目标有敌军(下令时/hourly检查) → 战斗中(移动减速×0.33)
  │   ├─ 守方败 → 加速移动 → 到达占领
  │   └─ 攻方败 → 撤退
  └─ 目标无敌军 → 正常移动 → 到达占领(非己方)
```

---

## 主循环顺序（每小时）
```
1. check_engagements    — 移动中师遇敌 → 开战
2. resolve_all_battles  — 战斗结算(撤退/歼灭/带溃/战斗胜利占地)
3. reinforce_reserves   — 预备队补位
4. advance_movement     — 行军推进 + 到达占领(战斗中的地块不占领)
5. recover_org          — org恢复
```
